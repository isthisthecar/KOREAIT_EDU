request : 사용자의 요구사항, 사용자가 보낸 데이터를 받음

response : 응답 ,사용자에게 계산된 결과를 보냄

express
    웹서버 구현시 http 모듈을 이용해서 구현
        --> 처음부터 끝까지 필요한 모든 기능을 직접 구현
    직접 모든 기능을 구현하려면 시간 및 자원 소모가 크기 떄문에
        express 모듈이 이미 웹서버의 대부분 기능을 미리 구현하였기 때문에
        간단하게 웹서버 기능을 사용할 수 있다.


    데이터를 주고 받을때에는 기본적으로 문자열로 주고 받음.


    express 모듈 설치방법 : npm install express
