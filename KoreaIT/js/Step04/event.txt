1. 이벤트 : 웹에서 일어나는 상황 (클릭, 키보드 자판을 눌렀을 때)

마우스
    onclick : 마우스를 클릭했을때
    ondblclick : 더블 클릭 했을때
    onmousedown : 마우스 버튼을 눌렀을 때
    onmouseup : 마우스 버튼을 눌렀다가 떼었을때
    onmousemove : 마우스 이동할 때
    onmouseover / onmouseenter : 마우스 포인터가 영역에 올라갔을때
    onmouseout / onmouseleave : 마우스 포인터가 영역에 벗어났을때
