h1~h6 : 제목태그
p   : 문단 태그
hr  : 가로 구분선 긋기
br  : 줄바꿈 태그
&nbsp; : 띄어쓰기
&copy; : 카피라이트 특수 문자

--글꼴 관련 ---
b , strong  : 글꼴 두껍게
i , em   : 기울임꼴
u  : 밑줄

<mark> : 배경색을 노랑색으로 채움
<blockquout>  : 인용문, 탭만큼 띄어쓰기
<pre> : 태그 내부에 있는 내용을 그대로 출력

<목록 태그 관련>
<ul> : 순번이 없는 목록 태그
<ol> : 순번이 있는 목록 태그
<li> : 목록태그의 항목을 나타내는 태그

태그의 종류는 크게 두가지로 나눔
inline 태그 : 내용 일부분을 디자인 내용에 해당하는 태그   --> 글자로 취급 --> 줄바꿈이 일어나지 않음
                --> b, i, em, u, mark, img, audio, video, iframe
block 태그 : 내용에 대한 영역을 잡는 태그  --> 문단(영역)으로 취급 --> 자동으로 줄바꿈이 일어남
                -->h1 ~ h6, ul, ol, p, pre, blockquout

<img>  : 이미지태그
    src = 경로 , alt = 대체 텍스트
<video> : 동영상태그
    src = 경로 ,
    poster = 동영상 썸네일 이미지 경로, 
    controls = 컨트롤러 표시, 
    loop = 반복설정

<audio> : 음악태그
    src : 경로
    controls = 컨트롤러 표시, 
    loop = 반복설정

<iframe> : 외부사이트 페이지 출력
    src : 경로
    frameborder = 테두리 두께

<a> : 다른 페이지 이동
    hreft = 이동할 경로 
    title = 툴팁내용
    target="_blank"  : 새탭에서 열기
